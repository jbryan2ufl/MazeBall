A reminder for how to create a new project:
	git init
	git branch -M main
	git remote add origin https://github.com/jbryan2ufl/MazeBall.git
	git remote add template https://github.com/jbryan2ufl/3D-Graphics-Template.git
	git fetch template
	git merge template/main --allow-unrelated-histories
	git add .
	git commit -m "Merged unrelated histories from template"
	git push origin main
